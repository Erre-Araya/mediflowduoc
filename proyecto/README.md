
# 🏥 MediFlow AI: la receta digital para tu centro de salud.

## 📌 Descripción del Proyecto

**MediFlow AI** es una plataforma digital de gestión y atención médica que utiliza inteligencia artificial para optimizar los procesos administrativos y la interacción con pacientes en centros de salud.

El sistema permite:

* Agendar citas médicas automáticamente
* Responder consultas frecuentes mediante un chatbot inteligente
* Gestionar información de pacientes
* Realizar teleconsultas
* Integrar canales de comunicación como WhatsApp

Su objetivo principal es mejorar la eficiencia operativa y reducir los tiempos de espera, brindando una mejor experiencia tanto a pacientes como a profesionales de la salud.

---

## 🚨 Problemática

Muchos centros de salud presentan dificultades como:

* Alta carga administrativa
* Largos tiempos de espera
* Sistemas poco integrados
* Comunicación ineficiente con pacientes

Esto genera errores, retrasos y una experiencia deficiente en la atención médica.

---

## 💡 Solución

MediFlow AI propone una solución tecnológica que automatiza procesos clave mediante inteligencia artificial, incluyendo:

* 📅 Agendamiento automático de citas
* 🤖 Chatbot inteligente para consultas
* 💻 Telemedicina
* 🗂️ Gestión centralizada de pacientes

## ⚠️ Limitaciones

* No entrega diagnósticos médicos directos
* No interpreta imágenes médicas
* Requiere conexión a internet
* Depende del uso activo por parte del usuario

## 🌎 Contexto del Proyecto

El crecimiento en la demanda de atención médica ha generado desafíos importantes en la gestión de centros de salud. Muchos procesos aún son manuales o poco eficientes.

La transformación digital, junto con tecnologías como la inteligencia artificial y la telemedicina, permite mejorar significativamente estos procesos.

**MediFlow AI** surge como una solución innovadora que busca modernizar la atención médica, haciendo los servicios más accesibles, rápidos y eficientes.

## 🛠️ Descripción Técnica

### 🧩 Arquitectura del Sistema

El sistema está basado en una arquitectura de 3 capas:

* **Frontend (Presentación):** Interfaz de usuario
* **Backend (Lógica de negocio):** Procesamiento, reglas del sistema e IA
* **Base de datos:** Almacenamiento de información

---

### 💻 Tecnologías Utilizadas

#### Frontend

* React
* JavaScript
* CSS

#### Backend

* Spring Boot
* Node.js

#### Librerías y Herramientas

* Vite
* React Router DOM
* Link por Meet (videollamadas) - N8N
* Generador de códigos QR
* API externa  (IA para chatbot)

---

### 🗄️ Base de Datos

* MySQL (MySQL Workbench)

Almacena:

* Pacientes
* Personal médico
* Citas
* Historial clínico
* Consultas del chatbot
* Registros de teleconsultas

---

### ☁️ Infraestructura

* Servidor de aplicación
* Servidor de base de datos
